# nouns-world-directory

Filterable directory pulling live data from a published Google Sheet.

## v9
- Mobile filters: dropdown with checkboxes on small screens; chip grid on md+.
- Monotone tag styling for chips and card badges.
- Edge doodles using images from `/public/images/accessories/` (random angles/sizes).

## Add your accessory images
Place any of these files under `public/images/accessories/`:
- accessory-1-noun-2.png
- accessory-1-noun.png
